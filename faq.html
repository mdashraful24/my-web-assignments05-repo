Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that represents the structure of a web page as a
tree of objects, allowing scripts like JavaScript to interact with and modify the HTML document dynamically.

How do you select an element from the DOM?

In JavaScript, you can select an element from the DOM using methods like document.getElementById() to select
an element by its ID, or document.querySelector() to select the first element that matches a specified CSS
selector (like a class, tag, or ID).

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is added to a parent element to handle events
from multiple child elements. Instead of attaching multiple event listeners to each child, the event bubbles up to
the parent, which captures and handles it efficiently.

How do you manipulate an element's attributes and styles using the DOM?

In the DOM, you can manipulate an element's attributes and styles to change how it looks and behaves dynamically.
To modify an element's attributes, such as id, src, or href, you can use methods like setAttribute() or directly assign
values using property accessors like element. id = "newId".
